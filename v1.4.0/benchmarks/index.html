    Benchmarks

    Here are some benchmark timings for tensors in 3 dimensions. For comparison the timings for the same operations using standard Julia Arrays are also presented.

    In the table below, a denotes a vector, A, As denotes second order non-symmetric and symmetric tensors and AA, AAs denotes fourth order non-symmetric and symmetric tensors respectively.

    OperationTensorArrayspeed-up
    Single contraction
    a ⋅ a1.241 ns9.795 ns×7.9
    A ⋅ a2.161 ns58.769 ns×27.2
    A ⋅ A3.117 ns44.395 ns×14.2
    As ⋅ As5.125 ns44.498 ns×8.7
    Double contraction
    A ⊡ A1.927 ns12.189 ns×6.3
    As ⊡ As1.927 ns12.187 ns×6.3
    AA ⊡ A6.087 ns78.554 ns×12.9
    AA ⊡ AA60.820 ns280.502 ns×4.6
    AAs ⊡ AAs22.104 ns281.003 ns×12.7
    As ⊡ AAs ⊡ As9.466 ns89.747 ns×9.5
    Outer product
    a ⊗ a2.167 ns32.447 ns×15.0
    A ⊗ A9.801 ns86.568 ns×8.8
